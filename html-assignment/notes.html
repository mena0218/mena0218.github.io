HTML Tags

Navigation tag [nav]
The tag that enables web users to go from the home page of a site to another page
Form tag [form]
Enables Web users to communicate between themselves and the server such as comments and uploading pictures.
Label and input tags [label] [input]
These are identifying tags helping to telegraph to web users what information to type in the form. Labels are for giving forms a name whereas inputs are used for the type of information that is needed to be written such as emails, telephone numbers, URL, passwords, etc.
Image tag [img]
Allows an image from the computer or cloud to be uploaded into the website. Their size and shape can be modified by width and height and by border radius
List tags [li] [ul] [ol] [dl] [dt] [dd]
This is useful for listing things, objects, themes or someone's information. They can be unorganized [ul], organized [ol] or descriptive [dl] [dt] [dd].
